## Como inicializar a aplicação:

**clone este repositório para seu pc:**
`git clone https://github.com/rafaelcitario/elixir_api.git`

**entre na pasta da aplicação:**
`cd elixir_api`

**Rode o comando:**
`mix deps.get`
